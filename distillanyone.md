GitHub 上有個神奇專案：[dot-skill](https://github.com/titanwings/colleague-skill)，上線沒幾天星星數直接破 15k，感覺作者應該已經在寫辭呈了。
你只要丟進去一個人的聊天紀錄或文件，它就能把對方變成一份 Agent Skill。不只是學講話口氣，是連腦袋裡那套思考流程、決策邏輯，甚至情緒起伏都一起打包帶走。
內建三種模式：同事版、親密關係版、公眾人物版。等於你可以做一個會開會的同事，或一個專門安慰你的老友（還不會已讀不回）。
支援各家 Coding agent、Hermes Agent、OpenClaw，一句話就能裝好——比裝同事還簡單。
最近大陸流行的一句話：「你的同事並沒有離開，他只是被蒸餾成 Token 的樣子！」
