# １）王道・透明感美人
フォトリアリスティックな高精細スタジオポートレート写真。20代前半の日本人女性、柔らかさと華やかさを兼ね備えた端正な顔立ち。奥行きのある二重まぶたの横に長い目元・外側に向かって緩やかに上がるエレガントなアウターコーナー・まぶたの幅は狭くコントロールされたクリーンなライン・虹彩にウォームブラウンのグラデーション、柔らかさの中に芯を感じる高い位置の角張ったピークを持つ極めて細く繊細なアーチ型の眉・眉頭は鼻筋寄り、内側から発光するようなルミナスなウォームアイボリー肌、鼻筋が一直線に細く通った完璧なストレートノーズ・ブリッジは剃刀のようにクリーン・小鼻のフレアは皆無、光を受けて輝く高く際立った頬骨、顎先に向かって優美に細くなるシャープなフェイスライン、ふっくらとした上唇が特徴のキューピッドボウ形リップ、口角がごくわずかに上がった穏やかな微笑み・目元にも温かさが宿る — メイク：アイライナーなし・上まつ毛根元のみワンコートの最小限マスカラ、シアーなシャンパンゴールドのアイシャドウウォッシュ・目頭にごく控えめなハイライト、シアーなピーチコーラルリップ・自然なツヤ仕上げ、頬骨の高い位置にほんのりアプリコットのパウダーブラッシュ、ツヤ肌グロウスキン仕上げ / 髪：ナチュラルなダークブラウンの鎖骨丈セミロング、ふんわりとしたカーテンバング・額の中央が透ける長さ、顎下からのレイヤー入り、全体にゆるやかなSカールウェーブ・空気を含んだエアリーな質感、顔周りのおくれ毛が頬にやわらかい影を落とす、ハニーブラウンの温かい光沢 / アクセサリー：耳たぶでさりげなく揺れるイエローゴールドの小さなフープピアス・直径1センチ未満 / オフホワイトのリネン素材オーバーサイズシャツ・第一ボタンまで開けてデコルテをほんのり見せる・袖を手首の上まで無造作にロールアップ / 片手で髪を耳にかける仕草の途中・指先が髪の束を耳の後ろへ送る自然な動きの一瞬 / シームレスなウォームトープのスタジオ背景、大型ソフトボックスによるラッピングライト・影をほぼ消した柔らかく均一な照明・肌のツヤを際立たせるキャッチライト
# ２）親しみスマイル美人
フォトリアリスティックな高精細スタジオポートレート写真。20代前半の日本人女性、親しみやすさと凛とした美しさが共存する顔立ち。奥行きのある二重まぶたの横に長い目元・シャープなアウターコーナーが水平方向にすっと伸びるクリーンなライン・まぶたの幅は狭く精密にコントロールされている・瞳が大きくキラキラとした光を含む、快活さの中に知的な鋭さを宿す高い位置の角張ったピークを持つ極めて細く繊細なアーチ型の眉・眉頭は鼻筋寄り、みずみずしく健康的なツヤのあるナチュラルベージュ肌・頬にごくわずかな自然な赤み、鼻筋が一直線に細く通った完璧なストレートノーズ・ブリッジは剃刀のようにクリーン・小鼻のフレアは皆無、高くて際立った頬骨、顎先に向かって優美に細くなるシャープなフェイスライン、キューピッドボウ形の自然な血色のコーラルピンクリップ、歯を見せない口角の上がったチャーミングな笑顔・目が三日月形に細まり目尻にごくわずかな笑いジワ — メイク：アイライナーなし・上まつ毛根元のみワンコートの最小限ブラウンマスカラ、シアーなピンクベージュのアイシャドウウォッシュ、シアーなコーラルピンクリップ・グロッシー仕上げ、頬骨にほんのりフレッシュピンクのクリームブラッシュ、ツヤ肌グロウスキン仕上げ・鼻先にごく自然な光 / 髪：ナチュラルなダークブラウンのミディアムボブ・顎下3センチの長さ、ふんわりとしたシースルーバング・おでこがほんのり透ける、毛先をランダムに外ハネさせたカジュアルなスタイリング、耳にかけず顔周りにナチュラルに落ちる毛束、軽やかなエアリーテクスチャー / アクセサリー：耳たぶにクラシックな直径6ミリのアコヤパール一粒ピアス / ラベンダーブルーの薄手クルーネックニット・体のラインを拾わないリラックスフィット / 片手で髪を耳にかける仕草の途中・ボブの毛束を指でそっと耳の後ろへ流す瞬間のキャプチャー / シームレスなライトベージュのスタジオ背景、正面やや上からの大型ソフトボックスによるフラットで均一なハイキーライティング・影を最小限に・瞳に丸いキャッチライト
