### 75casesClaude
現在大家最常煩惱的不是不想用AI，而是不知道要使用在什麼情境，Anthropic 官方網站整理了 75 個 Claude 真實使用情境，是他們實際做出來的案例展示，每一個都有具體情境、操作方式，還有用到哪些 Claude 功能。
用Claude幫忙處理網頁把全部 75 個案例整理出來，按類別分，每個都附上簡略說明，方便大家可以快速看看也許有跟自己工作相關案例，直接去網站看細節內容。

▍非營利組織（8 個）

1. 查看募款目標的實際需求
分析募款目標背後需要的具體資源投入和執行策略，讓組織領導人對目標有更清晰的認知。用 Claude 跑財務建模和情景分析，識別達成目標所需的時間、人力和資金。
使用功能：Analysis tool, Artifacts

2. 並列比較預算方案
在同一視圖中展示多個預算情境，幫助決策者理解不同方案的財務影響和取捨。Claude 可產出表格和圖表，清晰呈現各方案的成本效益分析結果。
使用功能：Artifacts, Analysis tool

3. 認識捐贈者保留的價值
計算和分析捐贈者生命週期價值，用數字說明為什麼保留既有捐贈者比開發新客更划算。Claude 做數據分析和視覺化，製作有說服力的論證供籌款和董事會決策用。
使用功能：Analysis tool, Artifacts

4. 檢視並完善你的變革理論
將組織的變革邏輯圖視覺化，檢查推論過程中的假設和薄弱環節。Claude 協助結構化組織目標、策略和預期成果之間的因果關係，確保邏輯站得住腳。
使用功能：Artifacts, Projects

5. 工作流程改進規劃
識別非營利組織內的效率瓶頸和冗餘流程，規劃實際可行的改進方案。Claude 可分析流程圖、成本結構和時間利用，提出具體建議。
使用功能：Projects, Analysis tool

6. 建立志工管理系統
建置追蹤志工資料、排班、工作分配和貢獻的管理系統。Claude 可生成資料庫應用或整合現有工具。
使用功能：Artifacts, Computer use

7. 分析募款績效
評估募款活動的成效，找出有效的募款策略和改進空間。Claude 分析捐贈資料、識別模式、提出優化建議。
使用功能：Analysis tool, Projects

8. 撰寫影響力報告
將組織的工作成果整理成有說服力的影響力報告，展現實際成效。Claude 協助故事化呈現和資料統計整理。
使用功能：Artifacts, Analysis tool

▍教育（9 個）

老師、學生、研究人員都在這個類別。有些案例也適合拿來做內部培訓。

9. 規劃課程進度表
在規劃教學進度時識別出時間緊張或資源受限的教學週，提前做好教學設計和準備。Claude 分析課程內容的複雜度和學習曲線，協助優化教學節奏並預留複習時間。
使用功能：Artifacts, Analysis tool

10. 比較不同的獎助金選項
評估多個獎助金方案的申請難度、金額、期限和使用限制，找到最符合需求的選項。Claude 整理不同來源的獎助資訊，進行多維度比較分析。
使用功能：Analysis tool, Artifacts

11. 邊學邊應用公式
學習數學或科學公式時同步進行實際應用和練習，透過具體例子加深理解。Claude 提供互動式例題演練、即時答疑和進階應用場景。
使用功能：Artifacts, Analysis tool

12. 整理文獻回顧，呈現核心辯論
將蒐集的學術文獻分類、標記和視覺化，清晰呈現學界的主要爭點、觀點分歧和研究演進。Claude 可生成文獻地圖、概念矩陣和辯論框架，幫助快速掌握研究領域全景。
使用功能：Projects, Artifacts

13. 在選擇讀論文前先圖表化資料
在提交數據分析或論文前，先用圖表和統計視覺化原始資料，驗證分析方法是否恰當。Claude 快速生成多種圖表格式和統計摘要，幫助發現異常值和資料特性。
使用功能：Artifacts, Analysis tool

14. 讓黑板課程活躍起來
將傳統黑板教學的概念轉化為互動式視覺內容或可運行的工具，增加課堂吸引力。Claude 幫助生成互動圖表、模擬程式或動態演示，讓抽象概念更具體。
使用功能：Artifacts, Computer use

15. 將解釋背後的機制視覺化
用圖表、動畫或互動工具展示科學原理或理論概念背後的運作機制和因果關係。Claude 根據教學目標生成合適的視覺表達方式，幫助學生從機制層面理解現象。
使用功能：Artifacts, Computer use

16. 將研究轉化成簡報
把研究成果、數據和分析結果組織成邏輯清晰的簡報內容。Claude 幫助結構化論點、設計圖表、撰寫標題和過渡句，製作專業水準的簡報。
使用功能：Artifacts, Analysis tool

17. 建立客製化課程教材
根據教學目標和學生需求，生成符合特定課程的教材、講義和練習題。Claude 可調整難度和呈現方式。
使用功能：Artifacts, Skills

▍個人使用（10 個）

19. 用 Skills 提升 Claude 設計能力

使用功能：Skills, Artifacts

20. 建置互動式圖表工具
開發可視化工具自動將複雜資訊轉換成圖表。Claude 生成互動式圖表程式、圖表庫包裝或資料儀表板。
使用功能：Artifacts, Computer use

21. 整理桌面檔案
根據檔案內容而不只是檔名，智慧分類和整理電腦檔案到合適的資料夾。Claude 掃描檔案並建議分類規則。
使用功能：Cowork, Computer use

22. 繪製理解地圖，找出學習缺口
視覺化個人的知識結構，找出學習的缺漏處，制定補強計畫。Claude 根據你描述的內容繪製概念地圖，識別知識薄弱環節。
使用功能：Artifacts, Projects

23. 多情境壓力測試財務計畫
針對個人或家庭的財務計畫，模擬不同的經濟情境，評估財務承受能力。Claude 建立動態財務模型，測試失業、醫療費用或房市波動的影響。
使用功能：Artifacts, Analysis tool

24. 用 Claude 挑選更貼心的禮物
根據收禮者的興趣、個人背景和偏好，提出更有意義的禮物建議，避免落俗套。
使用功能：Projects, Analysis tool

25. 建立客製的願望清單應用
建置個人的目標追蹤或願望清單工具，可自訂類別、優先級和進度追蹤。Claude 生成簡單的網頁應用。
使用功能：Artifacts, Computer use

26. 設計在地採集指南
製作在地特色野菜或食材的蒐集指南，結合季節性資訊、地圖和識別方法。Claude 生成圖文並茂的指南和安全提醒。
使用功能：Artifacts

27. 將訊息串轉成研究筆記
整理和轉化 Slack 對話、簡訊等非正式溝通為結構化的研究筆記。Claude 提取關鍵資訊並組織成可用的知識。
使用功能：Projects, MCP 整合

▍專業工作（17 個）

案例數量最多的類別。跨職能、跨工具的整合應用集中在這裡，也是 Cowork 和 MCP 用得最多的地方。

28. 製作品牌資產
根據品牌定位和視覺識別系統，快速生成適用於多個場景和媒介的品牌資產。Claude 可設計網頁圖形、社群貼文範本、簡報主題和行銷資料，確保品牌一致性。
使用功能：Artifacts, Skills

29. 將品牌指南包裝成 Skill
將組織的品牌準則、設計系統和溝通規範轉化為 Claude Skill，確保所有 Claude 輸出都符合品牌標準。這讓工作流程中的品牌一致性自動化。
使用功能：Skills, Projects

30. 從瀏覽器圖表和資料夾資料建立分析
整合網路上的圖表、本地資料夾和其他來源的資料，生成綜合分析報告。Claude 結合瀏覽器檢視和本地檔案存取，形成完整的資訊圖像。
使用功能：Claude in Chrome, Cowork

31. 建立跨工具的每日簡報
自動蒐集來自 Slack、Google Analytics、CRM 等多個工具的資訊，生成每日摘要和優先級提示。Claude 連接各工具並整理成一份簡潔的晨間簡報。
使用功能：MCP 整合, Projects

32. 用 Cowork 批量處理廠商資料
用 Cowork 自動化廠商評估和資訊整理流程，批量處理 RFP 回覆或廠商審查。Claude 存取本地資料並生成比較表和評估報告。
使用功能：Cowork, Analysis tool

33. 從多個工具匯聚洞察製作簡報
將 CRM、Analytics、Sheets 等工具的資料和洞察匯總，製作資料驅動的簡報。Claude 協調多個資料來源並視覺化關鍵訊息。
使用功能：MCP 整合, Artifacts

34. 補助金提案組裝流程
建立系統化的流程，高效整合資訊和文件，批量製作補助金提案。Claude 管理多個提案的進度，提供範本和內容建議。
使用功能：Projects, MCP 整合

35. 開發方案工具包
製作包含政策、執行步驟、模板和資源的完整方案工具包供執行參考。Claude 系統化組織內容並提供清晰的執行指導。

36. 方案資料視覺化
將方案運營的複雜資料用圖表、儀表板或互動視覺化呈現，便於監督和決策。Claude 設計合適的視覺化形式並生成實現方案。
使用功能：Artifacts, Analysis tool

37. 生成 AI 政策
根據組織的需求和風險考量，快速草擬 AI 使用政策和指南。Claude 提供現成框架並根據組織規模調整。
使用功能：Artifacts, Projects

自動彙整專案進度、里程碑、風險和燃盡圖，生成定期狀態報告。Claude 管理多個專案並產生一致的報告格式。
使用功能：Artifacts, MCP 整合

39. 分析使用者回饋中的模式
整合客服、調查、社群媒體等多來源反饋，識別核心主題和趨勢。Claude 自動分類反饋並提供優先級排序。
