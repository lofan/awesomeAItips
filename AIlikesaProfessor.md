1. Professor-Style Explanation Prompt
Use this to deeply understand any complex topic.
Prompt:
Act as a university professor in [Subject].
Explain [Topic] to me from beginner → advanced level.
Use:
- Simple analogies
- Real-world examples
- Step-by-step logic
End with a short summary and key takeaways.

2. “Explain Like a Curriculum” Prompt
Breaks a topic into structured lessons.
Prompt:
Design a mini curriculum to master [Topic].
Divide it into logical modules from fundamentals to advanced concepts.
For each module include:
- What I must understand
- Why it matters
- A simple explanation

3. Feynman Method Learning Prompt
Ensures real understanding.
Prompt:
Explain [Topic] in simple language as if teaching a 12-year-old.
Then identify any hidden complexities I should know.
Finally, provide a deeper expert-level explanation.

4. Concept Simplifier Prompt
Turns confusing ideas into clarity.
Prompt:
Break down this complex concept into:
- Core idea
- Supporting ideas
- Real-world analogy
- Practical example
Concept: [Paste Topic]

5. Step-by-Step Deep Dive Prompt
Perfect for technical subjects.
Prompt:
Teach me how [Concept/System] works step by step.
Start with fundamentals → mechanisms → advanced implications.
Avoid jargon unless explained clearly.

6. Compare & Contrast Prompt
Speeds up understanding through contrast.
Prompt:
Compare [Concept A] vs [Concept B].
Explain:
- Key differences
- When to use each
- Advantages & limitations
- Real-world applications

7. Socratic Tutor Prompt
Makes learning interactive.
Prompt:
Teach me [Topic] using the Socratic method.
Ask guiding questions to help me think critically before giving explanations.

8. Real-World Application Prompt
Connect theory to practice.
Prompt:
Explain how [Topic] is applied in real-world industries.
Include:
- Practical use cases
- Why it matters
- Example scenarios

9. Mental Model Builder Prompt
For fast understanding.
Prompt:
Build a mental model to understand [Topic].
Use:
- Visual-style explanation
- Simple analogy
- Key cause-effect relationships

10. Knowledge Tester Prompt
Turns learning into mastery.
Prompt:
- After explaining [Topic], create:
- 5 comprehension questions
- 3 application questions
- 1 challenge question
Then evaluate my answers.
